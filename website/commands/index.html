よく使うパターン

基本的なワークフロー

# 1. 設定初期化
pfscan config init

# 2. MCPサーバー追加
pfscan catalog install @modelcontextprotocol/server-time --id time

# 3. スキャン実行
pfscan scan start --id time

# 4. 結果確認
pfscan view          # イベントタイムライン
pfscan tree          # 階層構造
pfscan status        # システム状態

リアルタイム監視

# イベントをリアルタイム表示
pfscan view -f

# 特定コネクターのみ監視
pfscan view -f --connector time

# エラーのみ監視
pfscan view -f --errors

プロキシモード

# 全コネクターでプロキシ起動
pfscan proxy start --all

# 特定コネクターのみ
pfscan proxy start --connectors time,weather

# Claude Desktopから利用
# claude_desktop_config.json に追加:
{
  "mcpServers": {
    "proofscan-proxy": {
      "command": "pfscan",
      "args": ["proxy", "start", "--all"]
    }
  }
}

グローバルオプション

全コマンドで使用可能なオプション：

オプション 説明
-c, --config <path> 設定ファイルパスを指定
--json JSON形式で出力
-v, --verbose 詳細出力を有効化
-h, --help ヘルプを表示
-V, --version バージョンを表示
--no-color 色を無効化
--lang <code> 言語を指定 (en/ja)

コマンドエイリアス

ProofScanには複数のコマンドエイリアスがあります：

フルネーム エイリアス 説明
proofscan pfscan, pfs メインコマンド
pfscan shell psh シェル起動
pfscan view pfscan v イベント表示
pfscan tree pfscan t ツリー表示
pfscan scan pfscan s スキャン実行
pfscan status pfscan st ステータス表示
pfscan config pfscan c 設定管理
pfscan archive pfscan a アーカイブ管理
pfscan catalog pfscan cat カタログ操作
